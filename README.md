# 网约车web端

> 本项目是一个区块链项目的前端部分~

**技术选型**
+ vue.js
+ 前端路由: vue-router, 前端路由表 src/routes.js
+ UI框架: ant-design-vue
+ CSS预处理器: less
+ 前后端分离: /mock/path1_path2_path3.js
+ 请求封装: src/utils/request.js

**其他说明**
+ webpack-dev-server的 proxy 配置见 vue.config.js

**本地运行（使用Mock数据）**
npm run serve

**使用线上数据**
npm run serve:no-mock
